## Common Issues & Solutions

<AccordionGroup>
  <Accordion title="The client does not detect Roblox">
    - Make sure Roblox is **running and the game window is active** before starting `client.exe`
    - Run the client **as administrator**
    - Close and reopen both Roblox and the client
  </Accordion>
  <Accordion title="The client does not start">
    - Check your antivirus â€” some antivirus software may block the client
    - Add the client folder to the antivirus **exceptions list**
  </Accordion>
  <Accordion title="Error: Failed to connect">
    - Ensure your **internet connection** is stable
    - Disable VPN/proxy and try again
  </Accordion>
  <Accordion title="Scripts not loading or working incorrectly">
    - Make sure you are using **compatible scripts** for your game
    - Check if the game has updated - some updates may require script fixes
    - Acoustic script maker issue
  </Accordion>
  <Accordion title="Why loader doesnt working anymore?">
    - **Token expires in 7 days** - generate new client after expiration
    - Client becomes invalid **if new version is released**
    - Token becomes invalid **after generating new client**
  </Accordion>
  <Accordion title="File not found on server / Cant reach assembly website">
    - disable or enable **VPN/Proxy**
  </Accordion>
  <Accordion title="Version mismatch">
    - If there has been a recent update to the assembly, **download the new version of Roblox**
    - If the assembly has not yet been updated to your current version, **downgrade Roblox** to the previous version (if Roblox has been updated arbitrarily. If you want to avoid this, download [fishtrap](https://github.com/fishstrap/fishstrap))
  </Accordion>
</AccordionGroup>
