### get_alpha

TThe `ui.get_alpha()` function returns the alpha value of the main menu, represented as a floating-point number between 0 and 1. This function is specific to the main menu and does not apply to other interface elements such as the bar, chat, player list, or explorer. The value indicates the transparency level of the main menu, where 0 is fully transparent and 1 is fully opaque.

```lua
ui.get_alpha()
```

### get_position

The `ui.get_position()` function returns the x and y coordinates of the main menu position as a table with two numerical values. This function is specific to the main menu and does not apply to other interface elements such as the bar, chat, player list, or explorer.

```lua
ui.get_position()
```

### get_size

The `ui.get_size()` function returns the x and y dimensions of the main menu. This function is exclusively for obtaining the size of the main menu and does not apply to other interface elements like the bar, chat, player list, or explorer.

```lua
ui.get_size()
```

### frame_rate

The `ui.frame_rate()` function returns the current frame rate of the UI overlay. This value represents how many frames per second (FPS) are being rendered for the overlay, providing a measure of performance and fluidity for UI elements.

```lua
ui.frame_rate()
```

### 2 elements in a row

The elements are created from keybind and colorpicker, which is why it is currently not possible to create three elements in a row.

```lua
local cc = ui.new_checkbox("checkbox colorpicker")
local ck = ui.new_checkbox("checkbox keybind")

cc:colorpicker("colorpicker", 1, 0, 0, 1)
ck:keybind("keybind")
```

```lua
local lc = ui.label("label colorpicker")
local lk = ui.label("label keybind")

lc:colorpicker("colorpicker", 1, 0, 0, 1)
lk:keybind("keybind")
```

### example [ set_visible ]

```lua
local checkbox = ui.new_checkbox("Enable Elements")
local button = ui.button("Click Me")
local combo = ui.new_combo("Select Option", {"Option 1", "Option 2"})
local slider = ui.slider_float("Opacity", 0.5, 0.0, 1.0, "%.2f")
local slider_int = ui.slider_int("Count", 5, 1, 10, "%d")
local colorpicker = ui.colorpicker("Team Color", 1.0, 0.5, 0.2, 1.0)

local elements = {
    button, combo, slider, slider_int, multiselect, colorpicker
}

cheat.set_callback("paint", function()
    table.foreach(elements, function(_, v)
        v:set_visible(checkbox:get())
    end)
end)
```

### example [ callback ]

```lua
local checkbox = ui.new_checkbox("Enabled")
local button = ui.button("Click Me")
local combo = ui.new_combo("Select Option", {"Option 1", "Option 2", "Option 3"})
local slider = ui.slider_float("Opacity", 0.5, 0.0, 1.0, "%.2f")
local slider_int = ui.slider_int("Count", 5, 1, 10, "%d")
local multiselect = ui.multi_select("Select Items", {"Item 1", "Item 2", "Item 3"})
local colorpicker = ui.colorpicker("Team Color", 1.0, 0.5, 0.2, 1.0)
local textinput = ui.textinput("Custom Text", "Default value")
local keybind = ui.keybind("Aimbot Key")
local label = ui.label("This is a label")

cheat.set_callback("paint", function()
    --> checkbox
    if checkbox:get() then
        print('active')
    end

    --> button
    if button:get() then
        print('clicked')
    end
    
    --> slider_int / slider.float
    local slidered = slider:get()
    print("slider value:", slidered)

    --> combo / multiselect
    local selected = multiselect:get()
    for _, index in ipairs(selected) do
        print("leo corre pa rezotti zakite", index) -- returns a number, 0-based
    end

    --> colorpicker
    local colored = colorpicker:get()
    print("recolored:", colored.r, colored.g, colored.b, colored.a) 
    -- or print("Selected color:", colored)

    --> textinput
    local texted = textinput:get()
    print("changed text:", texted)

    --> keybind
    if keybind:get() then
        print("keybind active")
    end
end)
```
