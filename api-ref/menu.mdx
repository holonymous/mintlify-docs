### get_alpha

TThe `ui.get_alpha()` function returns the alpha value of the main menu, represented as a floating-point number between 0 and 1. This function is specific to the main menu and does not apply to other interface elements such as the bar, chat, player list, or explorer. The value indicates the transparency level of the main menu, where 0 is fully transparent and 1 is fully opaque.

```lua
ui.get_alpha()
```

### get_position

The `ui.get_position()` function returns the x and y coordinates of the main menu position as a table with two numerical values. This function is specific to the main menu and does not apply to other interface elements such as the bar, chat, player list, or explorer.

```lua
ui.get_position()
```

### get_size

The `ui.get_size()` function returns the x and y dimensions of the main menu. This function is exclusively for obtaining the size of the main menu and does not apply to other interface elements like the bar, chat, player list, or explorer.

```lua
ui.get_size()
```

### frame_rate

The `ui.frame_rate()` function returns the current frame rate of the UI overlay. This value represents how many frames per second (FPS) are being rendered for the overlay, providing a measure of performance and fluidity for UI elements.

```lua
ui.frame_rate()
```

### 2 elements in a row

The elements are created from keybind and colorpicker, which is why it is currently not possible to create three elements in a row.

```lua
local cc = ui.new_checkbox("checkbox colorpicker")
local ck = ui.new_checkbox("checkbox keybind")

cc:colorpicker("colorpicker", 1, 0, 0, 1)
ck:keybind("keybind")
```

```lua
local lc = ui.label("label colorpicker")
local lk = ui.label("label keybind")

lc:colorpicker("colorpicker", 1, 0, 0, 1)
lk:keybind("keybind")
```
